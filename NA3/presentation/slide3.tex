{\centerline{\LARGE\color{darkRed} The A3R Application Model}}
\vspace{1em}
{\Large\begin{itemize}
\item A3R Applications are self-contained, citable resources which 
can conform to modern resource documentation standards, such as the 
Research Object protocol.	
\vspace{.3em}
\item A3R Applications can use Hypergraph-structured metadata 
to describe data types, procedures, User Interface features, 
and inter-type relationships (for instance, the relation 
between data types and the types of GUI components which 
visualize them). 	
\end{itemize}}\vspace{6pt}
\end{minipage}}\\\\\\

\hspace{5pt}\fcolorbox{fcBoxColor}{white}{\begin{minipage}{.95\textwidth}%
\vspace{.8em}
{\centerline{\LARGE\color{darkRed} A3R Developer Tools}\vspace{.8em}}
{\Large\begin{itemize}
\item Hypergraph-based data modeling and serialization.
\vspace{.5em}  
\item Framework for building custom scripting, parsing, and data persistence engines.
\vspace{.5em}   
\item Enhanced support for applications specifically designed to access research data 
sets. \vspace{.5em} 
\item Convenient framework for sharing data 
among applications (to establish inter-application 
workflows) or between applications and cloud 
or web services (including leveraging NCN services).
 \vspace{1em}
 \end{itemize}}\end{minipage}}\end{minipage}}
